## Python pomodoro timer project


This will utilize the micropython link to the official raspberry page: https://www.raspberrypi.com/documentation/microcontrollers/micropython.html

More features will be coming

Components
- 1x Raspberry Pi Pico
- 1x OLED SSD1306 screen
- 1x Passive buzzer
- 1x Rotary encoder



Finished features 👏
  - Knobs basic function to scroll through the panels
  - Able to set time and save time
  - Set timer function
